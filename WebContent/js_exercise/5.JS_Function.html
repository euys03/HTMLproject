함수(Function) - JavaScript 함수의 종류

1. 선언적 함수

변환 값 선언없이 function 키워드만 이용하여 사용
function 함수명() { 소스코드 작성 }으로 작성해서 실행하는 방법


2. 익명함수

function에 함수명을 작성하지 않고 변수에 대입하는 방법
함수명 = function() { 소스코드 작성 }로 작성하여 실행하는 방법


3. 스스로 실행하는 함수(즉시 실행 함수, IIFE)

(function() { 소스코드 작성 })();로 작성


4. 함수의 전달인자와 매개변수


5. 함수의 리턴처리


6. 매개변수로 함수 전달 (feat. callback 함수)

익명함수를 매개변수로 활용
함수도 하나의 자료형이기 때문에 매개변수로도 전달 가능


7. 익명함수를 리턴하는 함수
