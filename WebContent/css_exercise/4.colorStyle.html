color 속성 : 텍스트의 색상을 지정할 수 있다.

선택자 { color : 색상영문이름 | 16진수숫자 | rgb값 | rgba값; }의 형식으로 사용
  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb로 지정
  • rgba로 지정

text-decoration 속성 : 텍스트에 줄을 긋거나 줄을 없앨 때 사용

선택자 { text-decoration : none | underline | overline | line-through; }
  • none
  • capitalize : 영문자의 첫글자만 대문자로
  • uppercase : 모든 영문자를 대문자로
  • lowercase : 모든 영문자를 소문자로

text-shadow : 텍스트에 그림자 효과를 줄 때 사용

선택자 { text-shadow : none | 가로거리 세로거리 번짐정도 색상; }의 형식으로 표현함
