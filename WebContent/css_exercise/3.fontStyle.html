1) font-size 속성 : 글씨의 크기를 표현할 때 사용

 - '선택자' + { font-size : 30px | pt | em | rem | %; } 형식으로 사용
  • 고정크기 px 테스트
  • 가변크기 em 테스트
  • 가변크기 % 테스트

2) font-weight 속성 : 두꺼운 글씨를 표현할 때 사용

 - '선택자' + { font-weight : normal | bold | bolder | lighter | 100 | 400 | 900; }
  • 원래 굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

3) font-variant 속성 : 작은 대문자로 변경할 때 사용

 - 선택자 { font-variant : 속성값; }의 형식으로 사용

4) font-style 속성 : 텍스트의 기울임을 지정할 때 사용

 - 선택자 { font-style : normal | italic | oblique; }
  • italic 기울임 글꼴
  • oblique 기울임 글꼴

5) font 속성 : 글꼴 관련 스타일을 한번에 지정할 때 사용

 - 선택자 { font : font-style font-variant font-weight 
    font-size/line-height font-family; }의 형식으로 사용

  • 글꼴 관련 스타일 테스트2
